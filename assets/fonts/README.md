# Fonts for design renderer

Place font files here for template-based rendering. Recommended:

- **Inter**: https://fonts.google.com/specimen/Inter (Inter-Regular.ttf, Inter-Bold.ttf)
- **Poppins**, **Montserrat**: optional

The renderer looks for:
- `Inter-Regular.ttf` / `Inter-Bold.ttf` (or `Inter.ttf`) under `assets/fonts/`

If no font is found, the system fallback (sans-serif) is used.
